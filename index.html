CPU

The CPU (Central Processing Unit) is a very important part in a computer. The CPU is the brains of the computer.


Motherboard

The Motherboard is a part of a computer where all of the parts connect to. Parts like the PSU (Power Supply), GPU (Graphics Processing Unit), etc connects to
the Motherboard.


RAM

The RAM (Random Accsess Memory) is used when the computer wants to accsess a file from the storage device temporarily. RAM can have different capacities up to 64 GB (For consumers)


SSD

The SSD (Solid State Drive. Is a fast storage unit. It could be found in many modern computers or laptops today. SSD can also have different capacities too.


GPU

The GPU (Graphics Processing Unit) is also a very important part in a computer. The GPU (Graphics Processing Unit) Renders all of the images that you see on your screen.


Case

As the name suggests, the case is where all of the parts are stored


CPU Cooler

The CPU cooler is used to cool down the CPU. The CPU cooler prevents the CPU from overheating.


PSU

The PSU (Power Supply Unit) is also a very important part of a computer. The PSU delivers power to all of the components in the computer.
